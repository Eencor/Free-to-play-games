Реализован интерфейс для сайта Free-To-Play Games:
    Основная станица получает данные с сервиса RpidApi и отрисовыает полученные игры с учетом выбранным размером страницы. 
    При клике по картинке открывается страница с игрой. 
    Страница с игрой включает в себя: постер игры, некоторые данные, карусель обоев и системные требования если они предусмотрены.  
    На каждую игу производится запрос но, данные об игре хранятся 5 минут и пока они хранятся рендер происходит без дополнительного запроса.
    Также реализован индикатор загрузки и страница ошибки.
